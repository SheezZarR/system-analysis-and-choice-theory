# Условная энтропия и кол-во информации.

Расчет условной энтропии и количества информации для двух совместных (связанных) случайных событий, которые возникают вследствие броска двух игральных костей:
событие А - значение суммы чисел на гранях игральных костей;
событие B - значение произведения чисел на гранях игральных костей.
Задание:
> Создать функцию task, которая не принимает аргументов и возвращает список, содержащий пять чисел (с точностью до второго знака):
>
> Выходные параметры:
> 1. H(AB) - энтропия двух связанных (совместных) событий;
> 2. H(A) - энтропия события А;
> 3. H(B) - энтропия события B;
> 4. Ha(B) - условная энтропия события B связанного с событием A;
> 5. I(A,B) - информация в событии A о событии B.
